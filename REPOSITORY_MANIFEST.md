# Repository Manifest

## Included

- `Programs/`: data construction, feature engineering, survival modeling, bootstrap, supporting analyses, and pipeline orchestration scripts.
- `Manuscripts/build_exhibits.py`: builds manuscript tables and figures from analysis outputs.
- `Manuscripts/check_docs.py` and `Manuscripts/verify_submission_artifacts.py`: local manuscript package checks. These expect local submission-admin files that are not part of the public source snapshot.
- `Manuscripts/generated/tables/`: aggregate CSV/XLSX tables used by the manuscript and supplement.
- `Manuscripts/generated/figures/`: PNG, SVG, and PDF exhibit files. Large TIFF submission exports are excluded.
- `README.md`, `DATA_AVAILABILITY.md`, `REPRODUCIBILITY.md`, `requirements.txt`, and `CITATION.cff`: reader-facing metadata for rerunning and citing the code.

## Excluded

- Raw source files and record-level analysis datasets.
- Licensed ASHP/UUDIS and Symphony Health data.
- Intermediate parquet/cache files and fitted prediction artifacts under `Data/analysis/`.
- Submission administration checklists, reference-audit working files, and DOCX submission packages.
- Local scratch directories, archived analyses, agent state, and submission DOCX files.
- API keys and credential files.

The excluded files are intentionally omitted because they are either regenerated by the pipeline, too large for a source repository, or restricted by data-use terms.
